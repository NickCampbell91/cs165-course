Remote access using a Microsoft Windows computer

Please see this link for information on how to access the Linux Lab from home. You will need to perform the following steps if you have a Windows machine:

  1. Download the tool called PuTTY http://www.chiark.greenend.org.uk/~sgtatham/putty/download.html
  2. Go to the lab and read the IP address (four numbers separated by periods) from any machine in the lab. They are 157.201.194.201 through 157.201.194.210. This will be be virtual machine you are accessing when using remote access
  3. Boot PuTTY and type in your IP address from step 2 and the port 215. You might want to save this session so you don't have to keep typing the numbers in.
  4. Select [OPEN]. After you specify your username and password, you are now logged into that machine.

Remote access using an Apple Macintosh computer

If you are on a Macintosh or a Linux computer, bring up a terminal window and type the following command:
ssh <username>@<ip> -p 215 -X2

If, for example, you want to connect to machine 157.201.194.230 and your username is "myUserName", then you would type

ssh myUserName@157.201.194.230 -p 215 -X2
