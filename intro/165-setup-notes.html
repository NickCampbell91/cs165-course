CS 165 F2F Course Setup Notes

This document contains a list of the steps to convert the online course to a Face to Face course.

Each semester, the course improvement department will change the due dates, etc for the next semester. Then, we can copy the reference course to be a F2F reference, make a few changes there, and then let each professor create their F2F section from that source.

  1. Copy the course to a F2F Reference section.
  2. Remove the following:
    1. Course Resources
  3. Add the following (From the F2F files sub-module):
    1. Course Resources
  4. Add to the gradebook:
    1. Team Final Exam Review
    2. Final Exam
  5. Adjust the weights of the gradebook to be as follows:
    1. Team Final Exam Review
    2. Final Exam
