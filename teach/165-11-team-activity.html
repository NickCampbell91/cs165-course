11 Teach : Team Activity - Templates

This activity is designed to be completed during class.

Overview

This activity will help you practice the syntax of class templates. You will create a vector-like container, which we'll call a bag of holding. This container will start with a fixed size, and automatically grow as needed. It will use a dynamically allocated array to actually store items.

Instructions

Divide your teams into groups of 2-3. You should all work together on one laptop, then make sure everyone in your team gets a copy of the work that you complete.

You will begin with the files in /home/cs165new/ta11. The provided files include: bag.h, bag.cpp, and ta11.cpp.

Core Requirements

  1. Create a bag of holding that only works with integers by supplying the following methods in bag.cpp:

    1. default constructor - Creates a bag that can hold 5 ints.

      The constructor should dynamically allocate an array of 5 ints:

      _data = new int[5];

      In your constructor, set the _capacity member variable to the number of items the bag can hold. (5 in this case).

    2. getCount() - Returns the number of ints in the bag.
    3. getCapacity() - Returns the number of ints the bag can hold.
    4. getItem(int index) - Returns the int at the specified index.
    5. addItem(int value) - Adds an int to the bag. Every time you add an item to the bag, you'll need to update the _currentAddIndex variable so that you know where to insert the next item.

    Use the code in ta11.cpp to test your progress.

  2. Change the code in addItem so that the bag automatically resizes to accomodate more than 5 items:

    1. Modify the addItem() method. If _size is equal to _capacity, create a new dynamically allocated array of ints twice the size of the previous one (don't forget to update _capacity) use a for loop to copy all of the items from the old array to the new array, and then delete the old array, then add the new item.

    Uncomment the code in ta11.cpp labeled as /*PART II*/ to test your progress.

  3. Modify the bag.h and bag.cpp to be a class template, making sure that you only replace the instances of int that are appropriate.

    Uncomment the code in ta11.cpp labeled as /*PART III*/ to test your progress.

Stretch Challenges

After completing the above steps. Make sure that everyone on the entire team is to this point and understands the material. Then, if you have time, move onto the following stretch challenges.

  1. Make it possible to search your bag of holding for a value. (Test with /*STRETCH I*/ in ta11.cpp
  2. Make it possible to delete from your bag of holding. (Test with /*STRETCH II*/ in ta11.cpp
  3. Add a destructor to your bag of holding to clean up your dynamically allocated array.

Submission

You do not need to submit your program (just make sure that everyone has a copy of it for their reference). Instead, answer the accompanying questions in the I-Learn quiz.
